# Previsão de Churn com Machine Learning 🤖

Este projeto utiliza Inteligência Artificial para identificar padrões de comportamento de clientes e prever possíveis cancelamentos de contrato (Churn).

## 🚀 Como funciona
O modelo utiliza o algoritmo **Random Forest** para analisar variáveis como tempo de contrato, valor mensal e interações com o suporte técnico.

## 📊 Tecnologias
- Python
- Pandas (Manipulação de dados)
- Scikit-Learn (Machine Learning)
